## Installation

```
git clone git@github.com:daenub/webpack-boilerplate
yarn
```

## Usage

### Development server

```bash
yarn start
```

### Production build

```bash
yarn build
```

### Deploy to GitHub Pages

This command will commit and push the content of the dist folder to the gh-pages branch.

```bash
yarn deploy
```
